Fenix UI Skeleton
=======

Basic structure for Fenix Platform UI projects


#Requirements

* [NodeJS](http://nodejs.org/)
* [NPM](https://npmjs.org/) 
* [Grunt](http://gruntjs.com/)


#Deploy

```bash
npm install
grunt
```
